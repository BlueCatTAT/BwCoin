平台无法追回损失，用户需自行承担损失。

1-2：录入个人信息 → 点击确认 → 显示保存成功即设置完成。

密码修改/重置

一：修改密码

1-1:点击账号 → 账户设置 → 修改。

1-2：输入账号 → 点击发送验证码 → 点击确定。

1-3：输入新密码 → 点击确认。

二：忘记密码重置

2-1：密码忘记重置，点击登录 → 忘记密码。

2-2：输入账号 → 点击发送验证码 → 点击确定。

2-3：输入新密码 → 点击确认。

出售USDT

1-1：点击法币交易 → 选择出售USDT入口 → 点击出售USDT。

1-2：选择出售方式 → 输出出售金额或数量 → 点击下单。

1-3：商家付款，交易完成。

购买USDT指引

一：如何购买USDT

1-1：点击法币交易 → 选择购买USDT → 点击向商家购买USDT

1-2：选择够买方式 → 输入买入金额或数量 → 点击下单

1-3：根据商家提供的付款信息，付款给商家并点击我已付款，点击确认

二：法币交易订单查询

2-1：点击订单记录可查询订单状态

2-2：可根据条件筛选状态类型

杠杆交易指引

一：杠杆交易如何下单

1-1：点击杠杆交易

1-2：选择想要做的币种类型

1-3：市价交易，操作步骤（选择倍数 → 输入手数 → 点击买入做多/卖出做空）

1-4：限价交易，操作步骤（输入挂单价格 → 选择倍数 → 输入手数 → 点击买入做多/卖出做空）

二：杠杆订单查询

2-1：点击币种入口 → 选择要查询的币种 → 点击杠杆持仓（可查询当前币种持仓）→ 点击交易记录（可查询当前币种的交易明细）
